Selenium has been designed for Windows, Macintosh, BSD and Solaris.

The core mode of Selenium should more than those, but the Remote Control mode depends on two things.

  1. The availablility Java (JDK 5.0+) for it to run.
  2. An understanding from the Selenium team of how to launch a browser from the command line.

Selenium's own build is tested against Windows, Mac and Linux. We reply on user feedback to notify us of problems for BSD and Solaris where applicable, but have experienced good compatibility over the years.

We're keen on integrating as many as possible - and looking for volunteers to help.
