Selenium Core and IDE uses a simple language based on a series of commands :

open /seach.html
type searchField Cheese
click submit
verifyTextPresent Cheddar

These prove peopular for non-professional programmers.

Selenium Remote Control (RC) uses real programming languages, like Java, Ruby, Python, C#, Perl, PHP :

(Java)

selenium.open("/search.html");
selenium.type("searchField", "Cheese");
selenium.click("submit");
selenium.verifyTextPresent("Cheddar");

Selenium-IDE can save scripts to one of the other languages in order to kick-start the development of full RC suites of tests. It is a good bridge from the world of business-analysts, customers, end-users and project managers to developers. Meaning the the former group find it easy to record the situation that represents a defect in the application, play it back, then pass it to the developers to fix.
